Github User Displayer


To search for a user, insert the username in the following search field.

You must insert the username which is different from the name. For instance you might look for my account by searching for my username which is LucaIta (my name on GitHub is Luca Italy).

Other username you might try are "Facebook" or "Elementary".

Once the user details are displayed, click the "Display user repositories" to check them out.
